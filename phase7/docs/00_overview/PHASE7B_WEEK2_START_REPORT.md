#### **ステップ2: 単一カラムROI分析** ← 次のタスク

**やること**:
- 334カラム（または68カラム）を1つずつテスト
- 各カラムのROI計算
- ROIランキング作成

**作成予定ファイル**:
- ⏳ `phase7/scripts/phase7b/single_column_roi_analysis.py`
- ⏳ `phase7/results/phase7b_roi/phase7b_single_column_roi_ranking.csv`

**ROI計算方法**:
```
例: カラム「agari_shisu」（上がり指数）

1. 各レースで上がり指数が高い馬トップ3を選ぶ
2. トップ3に100円ずつ賭ける（合計300円/レース）
3. 的中金額（払戻金）を計算
4. ROI = (的中金額 / 購入金額) × 100
5. 全レース（460,424行）で平均ROI計算

例: 
  - 購入金額: 300円 × 58,234レース = 17,470,200円
  - 的中金額: 20,000,000円
  - ROI = (20,000,000 / 17,470,200) × 100 = 114.5%
```

**所要時間**: 約2-3時間
