# Phase 7 ワークフロー - 中央競馬回収率特化システム

**作成日**: 2026-03-02  
**データ期間**: 2016~2025年（10年間固定）  
**目標**: 年間回収率（ROI）100%超達成

---

## 📊 前提条件

| 項目 | 内容 |
|------|------|
| **データソース** | JRA-VAN + JRDB（垣根を越えた組み合わせ可能） |
| **ROI基準** | 生の回収率100%（補正なし） |
| **データ期間** | 2016~2025年（10年間固定） |
| **特徴量** | 139次元（JRA-VAN）+ 60~80次元（JRDB）+ 追加調査 |
| **前日情報のみ** | ✅ 当日馬体重・リアルタイムオッズ不使用 |
| **自動投票** | ❌ なし（予測のみ） |
| **資金管理** | PAT×8%方式 vs Kelly 0.25 比較検証 |
| **出力形式** | TXT + CSV（ブラックボックス排除） |

### **Phase 7-B: ファクター単体ROI分析（2週間）**

**目的**: 各ファクターの生ROI（100%基準）測定

#### タスク
- [ ] 7-B1: ファクター単体ROI計算
  - データ期間: 2016~2025年
  - JRA-VANファクター: 139個
  - JRDBファクター: 60~80個
  - 計算式: ROI = (的中回数 × 平均配当) / 購入回数 × 100
  
- [ ] 7-B2: ファクターランキング作成
  - ROI ≥ 110%: 超優良
  - ROI 100~110%: 優良
  - ROI 90~100%: 平均的
  - ROI < 90%: 単体弱い（組み合わせで救済）
  
- [ ] 7-B3: 統計サマリー生成
  - ROI分布ヒストグラム
  - 平均ROI、中央値、標準偏差

#### 成果物

- `factor_ranking_report.txt`
- `factor_ranking_data.csv`
