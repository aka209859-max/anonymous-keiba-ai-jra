## 🔧 Phase 4-B 分析に基づく重み調整

### 調整の背景

Phase 4-B の特徴量重要度分析により、回帰モデルに以下の問題が発見されました：

| 問題点 | 詳細 |
|--------|------|
| **特徴量の極端な偏り** | Top 1 特徴量が 59.6% の重要度を占有 |
| **Top 3 の寄与度** | 82.8% (健全な基準: < 50%) |
| **Top 10 の寄与度** | 97.0% (健全な基準: < 80%) |
| **主要特徴量** | `prev1_distance` (59.6%), `past5_time_avg` (17.5%), `distance_change` (5.7%) |

### 重み調整の根拠

**回帰モデルの重みを減少** (30% → 20%):
- 距離依存が強く、他の要因 (騎手、馬場状態、ペース) が考慮されていない
- モデルの多様性不足のリスク

**二値分類モデルの重みを増加** (30% → 40%):
- Phase 3 でバランスの取れた特徴量構成を確認
- AUC 0.9414 の高精度

**ランキングモデルの重みを維持** (40%):
- NDCG@3 0.8670 の高精度
- 着順予測の中核を担う

## 🎓 学んだ教訓

### 1. データ漏洩の早期発見の重要性
- Phase 3 で AUC = 1.0 の異常を検出 → 即座に修正
- **教訓**: モデルが「完璧すぎる」場合は必ずデータ漏洩を疑う

### 2. 特徴量重要度の検証
- Phase 4-B で Top 1 特徴量が 59.6% を占める異常を発見
- **教訓**: モデルの精度だけでなく、**特徴量のバランス**も検証する

### 3. アンサンブルでの重み調整
- 単一モデルの弱点を、アンサンブルの重み調整で補完
- **教訓**: 完璧な単一モデルを目指すより、**補完的なモデルの組み合わせ**を重視する
