```python

    COUNT(*) as total_rows,
    COUNT(column1) as filled_column1,
    ROUND(COUNT(column1)::NUMERIC / COUNT(*) * 100, 2) as fillrate_column1,
    -- ... 全カラム
FROM table_name 
WHERE kaisai_nen BETWEEN '2016' AND '2025';  -- 年次カラムがある場合
```

### Step 4: 結果分析

- **充填率 <80%**: 除外
