## 🎯 Phase 7-A データセット設計

### ベーステーブル: jvd_se（483,981行）

```sql
SELECT 
    -- jvd_se: 70カラム（出走馬基本情報）
    se.*,
    
    -- jvd_ra: 62カラム（レース情報）を結合
    ra.kyori, ra.track_code, ra.tenko_code, ra.babajotai_code_shiba, ...,
    
    -- jvd_ck: 106カラム（調教情報）を結合
    ck.chokyo_time, ck.chokyo_hyoka, ...,
    
    -- jvd_um: 89カラム（馬マスタ）を結合
    um.heichi_honshokin_ruikei, um.shiba_ryo, um.shiba_omo, ...

FROM jvd_se se
LEFT JOIN jvd_ra ra ON se.kaisai_nen = ra.kaisai_nen 
                    AND se.keibajo_code = ra.keibajo_code
                    AND se.race_bango = ra.race_bango
LEFT JOIN jvd_ck ck ON se.kaisai_nen = ck.kaisai_nen 
                    AND se.keibajo_code = ck.keibajo_code
                    AND se.race_bango = ck.race_bango
                    AND se.umaban = ck.umaban
LEFT JOIN jvd_um um ON se.ketto_toroku_bango = um.ketto_toroku_bango

WHERE se.kaisai_nen BETWEEN '2016' AND '2025';
```

**期待される出力**: 483,981行 × 327カラム（70+62+106+89）
