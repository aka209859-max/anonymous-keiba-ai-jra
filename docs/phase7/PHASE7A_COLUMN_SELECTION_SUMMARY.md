## 🎯 Phase 7-Bでの活用予定

### 単体ROI分析（Week 2-3）

全334カラムについて：

1. **単体ROI計算**
   - 各カラムの上位20%の馬を購入した場合のROI
   - 基準: 生の回収率100%

2. **ランク分類**
   - S: ROI ≥ 110%（超優良）
   - A: ROI 100~110%（優良）
   - B: ROI 90~100%（平均的）
   - C: ROI < 90%（単体弱い）

3. **使用/不使用判定**
   - S/A/B: 確実に使用
   - C: 組み合わせで救済検討
